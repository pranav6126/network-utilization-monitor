# Network Utilization Monitor using Mininet

## Project Description
This project demonstrates a simple Software Defined Networking (SDN) 
monitoring setup using Mininet.  
It simulates a small topology and monitors traffic between hosts.

## Topology
3 Hosts connected to 1 Switch.

## Execution Steps
1. Run topology.py
2. Start Mininet network
3. Perform ping tests between hosts
4. Monitor interface statistics
